# Telegram-бот розыгрыша для автошколы

## Что делает бот
- `/start` запускает сессию и показывает 6 призов
- `/roll` бросает анимированный кубик Telegram (`??`)
- По числу кубика выдается соответствующий приз
- Бот просит номер телефона кнопкой `Поделиться номером`
- Контакты сохраняются в `leads.csv`
- Опционально отправляет заявку менеджеру в чат (`MANAGER_CHAT_ID`)

## Быстрый запуск
1. Установи Python 3.11+.
2. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Создай `.env` на основе `.env.example` и укажи токен:
   - `TELEGRAM_BOT_TOKEN=...`
   - `MANAGER_CHAT_ID=...` (необязательно)
4. Запусти бота:
   ```bash
   python bot.py
   ```

## Как получить `MANAGER_CHAT_ID`
- Добавь бота в нужный чат/группу
- Отправь туда любое сообщение
- Открой `https://api.telegram.org/bot<ТВОЙ_ТОКЕН>/getUpdates`
- Возьми `chat.id` и вставь в `.env`
